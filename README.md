# Collaborative Whiteboard Application

A dynamic, real-time collaborative whiteboard designed to bridge the gap between professional productivity and an enjoyable user experience. This application allows teams to brainstorm, visualize ideas, and collaborate seamlessly in a customizable environment.

## 🚀 Overview

This project is built on the philosophy that work should be engaging. While many professional tools feel clinical, this application offers a unique, mood-based UI system that allows users to switch between a vibrant, "cute" aesthetic and a sleek, professional "cool" theme.

## ✨ Key Features

### 🖌️ Creative Toolkit
* **Freehand Drawing:** Smooth, low-latency brush strokes for sketching and annotating.
* **Shapes & Text:** Insert geometric shapes and text blocks to structure your ideas.
* **Sticky Notes:** Digital post-its for quick brainstorming and categorization.
* **Color Palette:** A curated selection of colors to keep your board organized and visually appealing.

### 🔄 Advanced Canvas Manipulation
* **Object Control:** Move, resize, and layer elements with intuitive drag-and-drop interactions.
* **Dynamic Layering:** Send objects to the front or back to manage complex layouts.

### 👥 Real-Time Collaboration
* **Live Cursors:** See exactly where your teammates are working with labeled, real-time cursors.
* **Participant Avatars:** Visual indicators of who is currently active on the board.
* **Activity Indicators:** Stay updated on recent changes and team presence.

### 👻 Intelligent Assistance: The Ghost Helper
* **Interactive Support:** A dedicated chat interface accessible via a custom ghost icon. 
* **Navigation & Shortcuts:** The assistant provides instant information on tool usage, keyboard shortcuts, and navigation tips to streamline your workflow.

## 🎭 The Mood-Switcher (Dual-Theme)

The application features a unique **Mood Toggle**, allowing you to adapt your workspace to your mental state:
* **Playful Mode:** A high-energy, pink-themed UI designed to boost mood and make creative sessions feel fun.
* **Professional Mode:** A "Cool" theme that strips back the aesthetic for a focused, minimalist environment when deep work is required.
